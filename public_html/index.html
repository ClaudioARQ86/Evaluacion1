Sobre el Curso

Este curso está diseñado para todos los niveles, desde principiantes hasta usuarios avanzados.

Aprenderás a:

  • Crear y gestionar hojas de cálculo
  • Utilizar fórmulas y funciones
  • Analizar datos con tablas dinámicas
  • Automatizar tareas con macros

Contenido del Curso

Módulo 1: Introducción a Excel

Conoce la interfaz de usuario y las funciones básicas.

Módulo 2: Fórmulas y Funciones

Aprende a utilizar las fórmulas y funciones más importantes de Excel.

Módulo 3: Análisis de Datos

Descubre cómo analizar datos de manera efectiva utilizando tablas dinámicas y gráficos.

Módulo 4: Automatización

Automatiza tareas repetitivas con el uso de macros.
